- Extensões feitas principalmente para o Modelo de Ficha de Minaīchi RPG, meu Sistema Própio.

- As Extensões usam o Modloader https://github.com/LucasLeiteDeveloper/RPGp-Estilos como base.

- A Extensão marcada como "com-corrupção" consta com um sistema automático que informa sobre cada estágio de corrupção do personagem, sendo feito estreitamente pro modelo de ficha citado acima.

- A Extensão "sem-corrupção" não consta com esse sistema e pode ser usada normalmente para outras fichas.
